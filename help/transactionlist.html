Kontoumsätze

Die Liste zeigt die von der Bank gesendeten Umsatzdaten in zeitlicher Reihenfolge. Meistens werden die in den letzten drei Monaten ausgeführten Transaktionen gesendet. Mit der Auswahlbox oben rechts können Sie den Zeitraum, der angezeigt werden soll, bestimmen.

Wenn Sie eine Überweisung detailliert angezeigt haben wollen, klicken Sie zweimal (Doppelklick) auf die entsprechende Zeile. Es öffnet sich dann ein Formular mit allen zu dieser Transaktion vorhandenen Informationen.

Die Anzeigefelder für Zahlungsein- und ausgänge unterhalb der Liste zeigen jeweils die im angezeigten Zeitraum aufsummierten Zahlungen an. Den letzten von der Bank gemeldeten Kontostand finden Sie ganz unten in der Statuszeile.

Sonderfunktionen

Sie können die in der Liste angezeigten Daten drucken oder exportieren lassen. MoneyPenny unterstützt derzeit den Export in CSV und SQL Formate. Die exportierten Daten werden unverschlüsselt auf einem Wechseldatenträger (Diskette, USB-Stick) gespeichert. Sie können mit fast allen Tabellenkalkulations- oder Datenbankprogrammen gelesen werden.

Sie können aus den vom Konto abgegangenen Überweisungen automatisch Überweisungsvorlagen erstellen lassen. Das ist recht praktisch, wenn Sie MoneyPenny erstmalig in Betrieb nehmen. Sie sparen sich so das Eingeben der Daten für Ihre häufig vorkommenden Überweisungen. Die Vorlagen werden in der Vorlagenliste gespeichert. Dort sollten Sie anschließend von Hand die Vorlagen löschen, die Sie nicht verwenden wollen. Dies können zum Beispiel Geldautomatenbuchungen oder ähnliches sein.
