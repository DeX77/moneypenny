Liste der Kontobewegungen anzeigen und drucken



Eine weitere wichtige Funktion des HomeBankings, Umsätze eines Kontos abzuholen und einzusehen, finden Sie unter dem Menüpunkt "Kontoumsätze".

Sie haben dort Zugriff auf die drei Funktionen:
• Kontoumsätze abholen
• Umsätze anzeigen
• Umsätze drucken



Umsätze abholen

Klicken Sie auf Kontoumsätze -> Kontoumsätze abholen, um ein Dialogfenster zur Abholung der Umsätze aufzurufen. Sie wählen zunächst, ob Sie für alle Konten oder für ein bestimmtes Konto Umsätze abholen möchten. Zusätzlich können Sie angeben, ob die Daten nur aktualisiert werden sollen oder ob Sie Umsätze eines bestimmten Zeitraums einsehen möchten. Mit Klick auf OK wird ihre Anfrage entweder sofort bearbeitet oder in der Auftragsliste gespeichert, wenn Sie im folgenden Dialog entweder auf "Jetzt" oder "Später“ klicken.

Eine Tabelle mit den aktuellen Umsätzen wird nach Ausführung des Auftrags angezeigt.



Umsätze anzeigen

Klicken Sie auf Kontoumsätze -> Umsätze anzeigen oder alternativ auf den Schnellstartbutton "Rotes Formular" und Sie erhalten eine tabellarische Übersicht über die Umsätze für das aktuelle Konto. Oben rechts wird die Kontonummer angezeigt sowie eine Aufklappliste in der Sie einen bestimmten Zeitraum wählen können, um die Übersicht zu wahren.

Die tabellarische Übersicht umfasst eine sortierte Liste durchnummerierter Umsätze. Dabei werden Angaben zu Datum, Name, Verwendungszweck, Kommentar, Bankleitzahl, Kontonummer und Betrag angezeigt.

Unter der Tabelle finden Sie außerdem eine Zusammenfassung der Einnahmen und Ausgaben als Summe der Umsätze des gewählten Zeitraums. Sie haben darüber hinaus die Möglichkeit, über die beiden Knöpfe am unteren Rand diese Liste auszudrucken bzw. zu aktualisieren.



Umsätze drucken

Klicken Sie auf Kontoumsätze -> Umsätze drucken, um eine Druckvorschau für die aktuellen Umsätze anzeigen zu lassen. Die Tabelle entspricht dabei weitestgehend der aus der Funktion "Umsätze anzeigen". Es werden zudem das Datum und die Seitenzahl eingefügt.

Mit dem Knopf "Drucken" drucken Sie das Dokument sofort aus. Stellen Sie sicher, dass ihr Drucker eingeschaltet ist und Papier vorhanden ist. Mit dem Knopf "Druckoptionen anzeigen" können Sie weitere Druckoptionen anzeigen lassen, in denen Sie z.B. die Druckerschriftart verändern können.

Mit dem Knopf "Druckereinrichtung" gelangen Sie in einen weiteren Dialog, in dem Sie Einstellungen für den Drucker vornehmen können.

Der Knopf "Hilfe" gibt ihnen weitere Informationen zum Druckvorgang.

Mit dem Button "Knopf" können Sie auf Wunsch auch den Druck abbrechen, sofern die Daten noch nicht an den Drucker geschickt wurden..
