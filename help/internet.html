Einrichtung der Internetverbindung



Falls Sie über einen Router oder übers Netzwerk mit DHCP ins Internet gehen, so wird diese Verbindung normalerweise erkannt und automatisch verwendet. Eine Internetverbindung wird in der in der Statusleiste des Programms angezeigt, sobald Sie die Verbindung zum Beispiel während der Ersteinrichtung haben prüfen lassen. Schlägt der Verbindungsversuch fehl, so erhalten Sie eine entsprechende Meldung.

Prüfen Sie in diesem Falle zunächst, ob Sie mit Ihrem anderen Betriebssystem eine Internetverbindung herstellen können. Falls dies auch nicht geht, so liegt der Fehler wahrscheinlich an der Hardware. Meist ist eine Kabelverbindung unterbrochen. Funktioniert die Verbindung unter dem anderen Betriebssystem, so ist die Hardware in Ordnung. In diesem Falle stimmen die Einstellungen für das Knoppix Betriebssystem von MoneyPenny nicht. Wahrscheinlich müssen Sie noch Ihre Zugangsdaten mit den von MoneyPenny bereitgestellten Werkzeugen konfigurieren.

Sie finden unter dem Menüpunkt Einrichtung/Hardware die für die Einrichtung Ihrer Hardware (Drucker, Modem, ISDN, DSL, GPRS) und Provider-Zugangsdaten verfügbaren Werkzeuge. Die Knöpfe auf der Hardware-Seite funktionieren mit den auf der Live-CD (Knoppix) vorhandenen Programmen. Alternativ können Sie auch direkt die vom Betriebssystem bereitgestellten Werkzeuge verwenden. Die dort vorgenommenen Einstellungen werden beim Beenden von MoneyPenny gespeichert und beim erneuten Programmstart automatisch wieder geladen.

Wenn Sie MoneyPenny unter einem schon installierten Linux Betriebssystem verwenden, so richten Sie mit den dort verfügbaren Programmen die Internetverbindung ein. Falls erforderlich, aktivieren Sie die Verbindung von Hand, bevor Sie Aufträge an die Bank verschicken lassen.
