Einzelüberweisung

Im Fenster rechts sehen Sie nun ein Formular für eine einzelne Überweisung
Geben Sie die benötigten Daten ein und überprüfen Sie ihre Eingaben sorgfältig, bevor sie den Auftrag in die Auftragsliste einfügen lassen. Klicken Sie, sobald Sie fertig sind auf "Ausführen" um den Auftrag der Auftragsliste hinzuzufügen.
Bitte beachten Sie, dass Sie auch die Möglichkeit haben das Ausführungsdatum festzulegen. Wählen Sie dazu die Option: Zahlungen > Terminüberweisung Der Auftrag wird erst ausgeführt, wenn Sie "Aufträge senden" im Hauptmenü auswählen.
