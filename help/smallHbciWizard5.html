Überprüfung der Daten

Überprüfen Sie hier, ob alle Daten (auch die automatisch eingelesenen) mit den Daten übereinstimmen, die Sie von Ihrer Bank mitgeteilt bekommen haben. Falls Sie eine Chipkarte verwenden, beachten sie bitte, dass die Daten dort vollständig vorhanden sein müssen. Lassen Sie diese gegebenenfalls von Ihrer Bank nachtragen.
