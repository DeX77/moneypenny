Terminüberweisung

Im Fenster rechts sehen Sie ein Formular für eine einzelne Terminüberweisung.
Geben Sie die Daten ein und überprüfen Sie ihre Eingaben sorgfältig.
Bitte beachten Sie, dass Sie das Ausführungsdatum angeben müssen. Klicken Sie, sobald Sie fertig sind auf "OK" um den Auftrag der Auftragsliste hinzuzufügen. Der Auftrag wird erst ausgeführt, wenn Sie "Aufträge senden" im Hauptmenü auswählen.
