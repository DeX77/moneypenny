Kontoübersicht
Dies ist eine kurze Übersicht über die Daten des aktuell ausgewählten Kontos. Diese Daten (außer der Beschreibung) wurden so von Ihrer Bank übertragen.
Wenn Sie ein anderes Konto überprüfen wollen, wählen Sie "Kontodaten / Kontoauswahl" im Hauptmenü, oder benutzen Sie den Knopf unten links im Hauptfenster.
