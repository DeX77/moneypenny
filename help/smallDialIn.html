Internet Einwahl

Mit den Knöpfen auf dieser Seite rufen Sie die Knoppix-Programme zur Einwahl ins Internet auf. Sie müssen aber vorher die Hardware für die Internetverbindung einrichten.

Klicken Sie je nach Hardware auf DSL, Modem oder ISDN. Wenn Sie über ein LAN mit einem DHCP Server verfügen, wird dies normalerweise automatisch eingerichtet. Sollten Sie einen kleinen Router mit http-Bedienungsschnittstelle haben, so können Sie mit dem LAN-Knopf den Firefox Browser als Bedieneroberfläche starten. WLAN und GPRS sind noch im Versuchsstadium.
