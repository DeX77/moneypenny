Kontoumsätze abholen

Sofern Sie über mehrere Konten verfügen, können Sie aus der Liste ein Konto auswählen, für das Sie die Umsätze abholen möchten. Direkt darunter können Sie den Zeitraum auswählen, für den Sie die Daten abholen lassen möchten. Bitte beachten Sie, dass einige Banken grundsätzlich alle verfügbaren Daten senden.
