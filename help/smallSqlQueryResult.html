SQL-Abfrageergebnis

Hier erscheint das Ergebnis Ihrer Abfrage. Falls nichts angezeigt wird, überprüfen Sie bitte die Installation Ihrer MySQL Datenbank zum Beispiel mit PhpMyAdmin. MoneyPenny benutzt die Datenbank „dbmoneypenny“ mit dem User „moneypenny“, Passwort „moneypenny“ auf Ihrem lokalen Computer „localhost“.
