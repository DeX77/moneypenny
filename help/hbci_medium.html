HBCI Einrichtung Seite 4, Privater Schlüssel

Schlüssel selbst erzeugen

Die sicherste Methode, einen privaten Schlüssel (private key) für die Datenübertragung zu erhalten ist, ihn vom eigenen Rechner erzeugen zu lassen. MoneyPenny erzeugt eine Schlüsseldatei und speichert diese auf dem angegebenen Medium (Diskette oder USB-Stick). Der Name der Datei setzt sich aus der Benutzerkennung, die Sie eingegeben haben und der Dateiendung ".key" zusammen. Am besten erstellen Sie eine Sicherheitskopie von dieser Schlüsseldatei auf einer Diskette und legen die Diskette an einem sicheren Platz ab. (Disketten kann man sogar in einen Ordner einheften, wenn man die Schreibschutzsperre öffnet.)

Fertigen Schlüssel verwenden

Eventuell haben Sie eine private Schlüsseldatei(private key) von Ihrer Bank erhalten.
Verwenden sie nur private Schlüsseldateien, die Sie auf dem Postwege erhalten oder die Sie selbst auf einer Diskette abgeholt haben. Über das Internet (email) versendete private Schlüsseldateien sollten Sie auf gar keinen Fall benutzen! Siehe hierzu das Kapitel über Sicherheit.

Chipkarte verwenden

Haben Sie eine HBCI-Chipkarte vom Typ DDV (Sparkassenkarte) von Ihrer Sparkasse erhalten, so wählen Sie den Eintrag "Ich habe eine Chipkarte..." aus. RSA-Chipkarten (VR-Networld) werden von MoneyPenny derzeit noch nicht unterstützt. Für Ihre Volksbank müssen Sie eine Schlüsseldatei verwenden. Wählen Sie den für Sie zutreffenden Knopf in der oberen Gruppe aus.

HBCI-PIN-TAN

Für das PIN-TAN Verfahren wird für die Anmeldung nur eine PIN und eine beliebige Kontonummer des Benutzers gebraucht.

Wo ist der Schlüssel?

Wenn Sie einen Schlüssel von Ihrer Bank erhalten haben, befindet er sich wahrscheinlich auf einer Diskette. Fragen Sie Ihre Bank nach dem Namen der Schlüsseldatei.
Bei einer Chipkarte befindet sich der Schlüssel auf der Karte.
Wenn Sie einen Schlüssel mit MoneyPenny erzeugt haben, sichert MoneyPenny den Schlüssel als Benutzernummer.key Datei nach Ihrer Wahl auf Ihrer Sicherheitsdiskette oder Ihrem USB-Stick. Der private Schlüssel sollte grundsätzlich nur auf einer Diskette oder einem USB-Stick gespeichert werden. Wählen Sie in der unteren Box im HBCI-Assistenten das entsprechende Medium aus. Da MoneyPenny unter Linux läuft, sind die Pfadbezeichnungen für Windows-Kenner etwas ungewohnt. Es gibt keine Laufwerksbuchstaben. Das Dateisystem beginnt mit einem /.
Die Pfade sind unterschiedlich, je nachdem, ob MoneyPenny von der Life-CD oder unter einem normalen Linux läuft. MoneyPenny schlägt den Pfad automatisch vor. Drücken Sie auf den Knopf "Schlüsseldatei von Hand suchen" und markieren Sie in dem Dialogfenster, das sich dann öffnet, die ".key" Datei mit Ihrem Schlüssel. Normalerweise sollte MoneyPenny schon das richtige Verzeichnis geöffnet haben. Sie können aber auch von Hand das Verzeichnis wechseln. Dazu haben wir in der folgenden Tabelle die wahrscheinlichen Pfade eingetragen.
Hinweis für Linux-Kenner: Es ist zwar möglich, den Schlüssel mit MoneyPenny auf Ihre Festplatte zu schreiben, Sie sollten dies aber aus Sicherheitsgründen unterlassen.
