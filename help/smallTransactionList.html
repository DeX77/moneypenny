Umsätze
Hier sehen Sie eine Übersicht über die Umsätze ihrer Konten.
Sofern Sie über mehrere Konten verfügen, können Sie oben rechts über der Liste ein Konto auswählen, für das Sie die Umsätze angezeigt haben möchten.
Direkt darunter können Sie einen Zeitraum auswählen, den Sie anzeigen lassen möchten.
Unterhalb der Liste gibt ihnen Money-Penny eine Übersicht über ihre Ein- und Ausgaben während dieses Zeitraums.
