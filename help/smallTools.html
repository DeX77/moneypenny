Werkzeuge

Bitte beachten Sie, dass die auf dieser Seite zur Verfügung stehenden Werkzeuge ohne vorhergehende Prüfungen die entsprechenden Kommandos ausführen. Sie sollten sie also nur benutzen, wenn Sie genau wissen, was Sie tun.

Diese Seite enthält eine Reihe von Werkzeugen für unvorhergesehene Probleme. Sie können hier probeweise eine Internetverbindung zu einem Server Ihrer Wahl aufbauen lassen, um Ihre Internet Einstellungen ohne das HBCI-Protokoll zu prüfen.
Die manuelle Auftragserstellung ist dazu gedacht, die Reaktion des Bankservers auf die standardisierten Aufträge herauszufinden.
Die Hardware-Konfiguration sollten Sie mit den vom Betriebssystem bereitgestellten Werkzeugen vornehmen, falls MoneyPenny nicht von der CD läuft oder die von uns ausgewählten Tools nicht zum Ziel führen. Speichern Sie anschließend Ihre neue Hardware-Konfiguration für die MoneyPenny-CD mit „Knoppix Konfiguration sichern“ auf Ihrem Wechselmedium ab.
